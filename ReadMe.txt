# Kraken ROV Camera Interface

## What this does
- Streams 3 cameras (front, gripper, rear) to a web interface
- Auto-starts on boot
- Runs on Raspberry Pi

## Requirements
- Raspberry Pi (any model)
- 3 USB cameras
- USB hub (if needed)

---

## Quick Setup (One command)

```bash
git clone your-repo-url
cd your-repo-name
chmod +x setup.sh
./setup.sh
